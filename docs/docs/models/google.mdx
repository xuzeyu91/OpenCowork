# Google Gemini

Gemini 适合超长上下文、多模态输入和大量文档处理任务。

## 配置入口 / Setup

进入 **设置 → AI 提供商 → Google**，通常会配置：

| 配置项 | 值 |
| --- | --- |
| 协议 | `openai-chat` |
| Base URL | `https://generativelanguage.googleapis.com/v1beta/openai` |
| API Key | Google AI Studio 的 key |

## 常见模型 / Common model examples

| 模型 | 场景 |
| --- | --- |
| `gemini-2.0-flash` | 快速、多模态 |
| `gemini-2.5-pro` | 更强能力 |
| `gemini-2.0-flash-thinking` | 推理 / thinking |

## 特点 / Characteristics

- 超长上下文窗口
- 图片 / 视频 / 音频等多模态能力
- 适合长文档和复杂上下文整理
- 通过 OpenAI-compatible 接口接入 OpenCowork

## 什么时候选 Gemini？ / When to choose it?

- 你要处理超长文档
- 你要混合图片和文本
- 你希望一个模型能扛住比较长的上下文
