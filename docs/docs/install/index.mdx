# Install

OpenCowork 是 Electron 桌面应用。发布版本从 GitHub Releases 获取；本地开发需要 Node.js 18 或更高版本，并按 Electron 版本重建 native modules。

<SignalGrid>
  <Signal title="release" value="Desktop first">
    面向日常使用优先选择安装包，避免本地 native module 环境差异。
  </Signal>
  <Signal title="source" value="Node.js 18+">
    从源码运行需要安装依赖、执行 postinstall，并启动 Electron + Vite。
  </Signal>
  <Signal title="data" value="Local only">
    运行时数据默认写入 `~/.open-cowork/`，不要提交到仓库。
  </Signal>
</SignalGrid>

<Callout type="warn" title="Windows native modules">
Windows 上 `node-pty` 会被 postinstall 脚本跳过；`better-sqlite3`、`@jitsi/robotjs`、`ssh2` 等 native modules 仍需要按 Electron 版本 rebuild。
</Callout>

## Desktop release

<CommandGroup title="Download release" description="选择与你的系统匹配的安装包。">
  <CommandItem label="Latest release" command="https://github.com/AIDotNet/OpenCowork/releases/latest" note="Windows 使用 .exe installer，macOS 使用 .dmg，Linux 使用 AppImage 或 .deb。" />
</CommandGroup>

## From source

<CommandGroup title="Run locally" description="用于开发、调试或验证当前仓库。">
  <CommandItem label="Clone" command="git clone https://github.com/AIDotNet/OpenCowork.git" />
  <CommandItem label="Install" command="cd OpenCowork && npm install" />
  <CommandItem label="Rebuild" command="npm run postinstall" note="根据当前 Electron 版本处理 native modules。" />
  <CommandItem label="Start app" command="npm run dev" note="首次启动 Vite 需要编译较多 React 模块，通常比后续启动慢。" />
</CommandGroup>

## Build installers

<CommandGroup title="Build pipeline" description="提交前建议先跑类型检查，再执行构建。">
  <CommandItem label="Type check" command="npm run typecheck" />
  <CommandItem label="Build app" command="npm run build" />
  <CommandItem label="Windows" command="npm run build:win" />
</CommandGroup>

## Local data directory

OpenCowork 的本地数据默认保存在：

```text
~/.open-cowork/
```

| Path | Purpose |
| --- | --- |
| `data.db` | SQLite 数据库 |
| `config` | 本地配置 |
| `agents/` | 用户可编辑 agents |
| `commands/` | 用户命令 |
| `prompts/` | 提示词 |

## Docs site

文档站在 `docs/` 目录中，是独立的 Fumadocs + Next.js 应用。

<CommandGroup title="Run docs" description="只开发文档站时不需要启动 Electron 主应用。">
  <CommandItem label="Enter docs" command="cd docs" />
  <CommandItem label="Install deps" command="npm install" />
  <CommandItem label="Start docs" command="npm run dev" note="默认地址为 http://localhost:3000。" />
</CommandGroup>
